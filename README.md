# Study Aura / HackathonEducAI

Study Aura is an AI-powered educational platform that transforms any source into structured study material.

## 🌟 Features
- **Smart Ingestion**: Process PDFs, YouTube videos, websites, and images.
- **AI Studio**: Generate Quizzes, Flashcards, Mind Maps, and Audio Overviews.
- **Gamification**: Earn XP and level up as you study.
- **Privacy First**: Your data stays in your Supabase instance.

## 🚀 Quick Start
See [QUICKSTART.md](./QUICKSTART.md) for a 5-minute setup guide.

## 🏗️ Technical Stack
- **Frontend**: React 19, TypeScript, Tailwind CSS 4.
- **Backend**: n8n, Supabase.
- **AI**: Gemini, Groq (Whisper/Llama), Mistral.
- **Infrastructure**: Docker Compose.
